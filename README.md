# Binance Watcher

# Requirements

- Python3
- Binance API KEY
- Telegram bot API KEY

## Install dependencies

```sh
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```

## Set API KEYS:

```sh
export API_KEY_READ_ONLY=...
export SECRET_READ_ONLY=...
export API_KEY_TELEGRAM=...
```

# Run the bot:

```sh
. .venv/bin/activate
python main.py
```
